# Contenerización de la Aplicación Web - Gestión de Películas

Este repositorio contiene el backend y el frontend del sistema de gestión de películas, junto con su respectiva contenerización usando Docker y Docker Compose.

## Estructura:
- Backend (Node.js + Express + MongoDB Atlas)
- Frontend (React)
- Dockerfile individual para cada módulo
- Orquestación con docker-compose.yml
